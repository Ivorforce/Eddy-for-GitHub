{# Reusable Jinja macros. Imported via {% from "_macros.html" import author_chip %} from templates that need it. Keeping macros here (rather than inlining them per template) means the SVG paths and class wiring live in exactly one place — change the icon once, every call site updates. #} {# Inline author = badge icon + login text. Mirrors the badge taxonomy in app/web.py:_AUTHOR_BADGE + _author_badge_class. Used in the popover timeline (per-event author, coalesced "comments by alice, bob" lines, etc.) — distinct from the richer rendering in _row.html's Repo column, which adds a tracking- link + popover and stays inline because of the per-row anchor wiring. Args: login — GitHub login string. Required. badge_class — '' / 'self' / 'member' / 'first-time' (or anything that maps to a .icon-* class in base.html). Empty string falls through to the muted generic icon. assoc — optional raw author_association string ('OWNER', 'CONTRIBUTOR', 'NONE', etc.) for the icon's title tooltip. Pass empty / None to suppress the tooltip. #} {% macro author_chip(login, badge_class='', assoc='') %} {%- if badge_class == 'self' -%} {%- elif badge_class == 'member' -%} {%- elif badge_class == 'first-time' -%} {%- else -%} {% set assoc_tip = assoc|lower if assoc and assoc != 'NONE' else '' -%} {%- endif -%} {{ login }} {%- endmacro %}
